
## Install software

**Update the pi**

    sudo apt-get update
    sudo apt-get upgrade
    sudo apt-get update && sudo apt-get upgrade -y

**Install the dependencies**

    sudo apt-get install hostapd isc-dhcp-server isc-dhcp-server-ldap
    sudo apt-get install iptables-persistent
